📖 使用說明

1. 取得 API Key

前往 Google AI Studio 建立你的 API Key。

2. 選擇 API 模式

Chat Completions — 適用於 Google Gemini、Claude 等透過 chat 回應生成圖片的模型。
Images API — 適用於 OpenAI DALL·E 等使用標準 /v1/images/generations 端點的服務。

3. 設定 Base URL

如果使用 Google Gemini API，可以留空（自動使用預設值）。如果使用 OpenAI，填入 https://api.openai.com/v1。

4. 選擇模型

Chat 模式預設 gemini-3-pro-image-preview；Images 模式預設 dall-e-3。

5. 測試連線

儲存後點擊「測試連線」確認 API Key 可正常使用。
